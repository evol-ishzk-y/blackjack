## 💻 プロジェクト概要

標準的なブラックジャックのルールをコマンドライン上で再現したゲーム。

* **バージョン管理:** 　GitHub

---

## 📐 システム設計

[カードのルール]
2〜10のカード: そのままの数字で足す（例：heartの7 は 7点）

J, Q, K (11, 12, 13): すべて 10点 として足す

A (1): 今回はまずシンプルに 1点 として計算する

* Program.cs (メイン)

* Player.cs (プレイヤー・ディーラー)

* Deck.cs (山札)

* Card.cs (トランプ)

---

## 💡 工夫した点・技術的アピール
* ディーラーの思考の作成: 17以上になったらカードを引くのをやめる
* 入力バリデーション： yesとno、大文字を入力しても処理する機能。それ以外もう一度入力させるように作成

## 🔮 今後の展望

改善したい点
* 入力の仕方によってコマンドの動きが予定と違う動作をしたので、様々なパターンを予測できるコードを入れるようにしたい
* 定数などを活用して数字に意味を持たせるようにして、コードの視認性を高める意識をしたい

挑戦したい機能
* 単体テストの実装
* 11～13も10で加算するのではなく数値で計算する
* チップ機能：掛け金を数字で指定して、勝ち負けによってチップを移動させる
* Windows Formアプリケーションで視覚的に表示できるようにしたい
